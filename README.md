# Mini Zero Desktop Pet

A small Python desktop pet that puts a mini Zero from Code Geass on your screen

The character stays always on top of the other windows, can be moved around the screen and has a simple animation

## Features

- Transparent window
- Always on top of other applications
- Mini Zero animation
- Drag the character with the mouse
- Close with right click or ESC key

## Requirements

You need Python 3 and the Tkinter library.

## How to run

1. Download or clone this repository
2. Put the image files in the same folder as `main.py`
3. Run `main.py`

## How it works

The program uses Tkinter to create a transparent window and displays two different images to make a simple animation

The `after()` function is used to change the images every few milliseconds

## Controls

- Left click + drag: move Zero around the screen
- Right click: close the program
- ESC: close the program
